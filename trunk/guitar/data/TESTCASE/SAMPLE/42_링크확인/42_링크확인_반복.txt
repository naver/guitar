;데이터 파일을 사용하여 반복적으로 팝업으로 열리는 링크의 정상 작동을 확인

IE를 생성한다. 

http://map.naver.com 접속한다.

; 첫번째 데이터 부터 사용
"$팝업링크목록=1"로 참조설정한다.


$i=1 설정, $i=9999 반복 한다. {

	$테스트링크정보=$팝업링크목록 설정한다.
	; 데이터가 끝일 경우 무한 반복문 탈출
	"$테스트링크정보=END" 같으면 $i=9999 설정한다.

	$i=9999 같지않으면 {
		; 첫번째 칼럼 데이터를 분리하여 저장
		"$클릭링크=$테스트링크정보,1"로 부분설정한다.
		"$확인링크=$테스트링크정보,2"로 부분설정한다.

		; TAG 방식으로 해당 클릭링크 클릭하고 확인링크가 있는 창으로 찾아 포커스를 이동
		"$클릭링크" 클릭한다. , 
		"$확인링크" 선택한다.

		; 현재화면 캡쳐
		현재화면을 캡쳐한다.

		; 팝업으로 열려진 (현재)창을 종료하고 원래 창으로 포커스를 이동하고
		현재 창을 종료하고  "$클릭링크" 선택한다.

		; 다음 테스트링크 정보를 가져옴
		$테스트링크정보=$팝업링크목록 설정한다.

	}
}

현재창을 종료한다.
